# Changelog

## 1.0.12+13 - 2026-03-30

### 新增
- 新增首页「管理课表」入口，支持导入 `.ics` 课表文件并用于点名自动匹配。
- 新增课表导入引导说明，内置 WakeUp 导出 iCal 的文字步骤与示意图入口（`assets/step1`、`assets/step2`）。
- 新增课表导入失败兜底：文件选择器不可用时自动回退导入内置 `class.ics`。

### 优化
- 优化课表展示：按“同课程 + 同时间 + 同地点”合并重复项，并展示多段周次（如 `第1-8周、第11-13周`）。
- 优化启动前引导页，移除底部分页指示点，界面更简洁。

### 移除
- 移除首页 FLAC 音频彩蛋及相关连击播放逻辑。
- 移除彩蛋音频预下载任务、配置项与服务代码。
- 移除 `audioplayers` 相关依赖与文档中的彩蛋音频配置说明。

### 发布信息
- 版本号：`1.0.12+13`
- 构建类型：Android Release APK
- 构建产物：`build/app/outputs/flutter-apk/app-release.apk`

## 1.0.10+11 - 2026-03-30

### 新增
- 新增启动前双页引导流程：权限授权页 + 协议确认页，强化首次使用的引导体验。
- 新增协议阅读完成门槛：需分别进入《用户协议》《隐私政策》并滚动到底，才可勾选同意。
- 新增协议阅读状态提示（已读到底/未完成）与阅读进度引导文案。
- 新增历史详情与学生详情的 AI 猫娘锐评能力（DeepSeek），支持结果本地缓存以减少重复消耗。
- 新增“本学期 / 全部记录”筛选能力（本学期起始时间：2026-03-01），覆盖历史与统计相关页面。

### 优化
- 合并“权限提醒 + 协议同意”为单一向导流程，并加入底部分页指示点。
- 调整权限申请时机：定位权限改为用户主动点击后再请求，不再在应用启动时自动申请。
- 优化权限页按钮布局与状态文案，解决重叠与对齐问题。
- 优化底部导航栏交互与过渡动画：选中态更简洁、切页与隐藏动画更顺滑。
- 优化历史详情页 AI 面板交互：支持悬浮、收起、滚动时隐藏、输入筛选时自动避让。
- 优化历史页筛选入口为右上角下拉菜单，并适配 Material You 圆角风格。
- 优化学生详情页 UI，自适应不同设备尺寸，统计块更紧凑并支持状态维度下钻列表。
- 优化更新包与缓存管理策略，降低临时文件占用风险，避免影响 token 与未同步本地数据。

### 修复
- 修复协议页使用 `Navigator` 打开文档时的上下文异常，改为根层叠加文档覆盖层。
- 修复多处动画与布局问题（含按钮重叠、弹窗留白、底部栏收起不顺滑、局部溢出等）。
- 修复 `FaIconData` 与 `IconData` 类型不匹配导致的编译报错。
- 修复曲线成员不可用导致的编译错误（替换为可用曲线）。
- 修复设置页主题色选择处过时 API 使用（`Color.value` -> `toARGB32()`）。

### 发布信息
- 版本号：`1.0.10+11`
- 构建类型：Android Release APK
- 构建产物：`build/app/outputs/flutter-apk/app-release.apk`
